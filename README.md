# CulinaryConnect

CulinaryConnect is a full-stack recipe sharing platform built with a **Node.js/Express** backend and a **Vanilla JS** frontend. It features secure authentication, user profile management, and a strictly validated recipe creation system.

## Key Features
* **User Authentication**: Secure Login/Register system using JWT.
* **Profile Management**: View and update user credentials (Username & Email).
* **Recipe Feed**: Dashboard to browse latest culinary creations.
* **Strict Validation**: 
    * Titles and Ingredients: Only alphanumeric characters and spaces allowed.
    * Instructions: Must be a strictly numbered list (e.g., "1. Step").

## Tech Stack
* **Frontend**: HTML5, CSS3, JavaScript (ES6+).
* **Backend**: Node.js, Express.
* **Database**: MongoDB with Mongoose.
